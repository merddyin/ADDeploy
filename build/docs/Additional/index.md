## Installation

ADDeploy is not currently available on the PowerShell Gallery.

To Inspect:

```powershell
Save-Module -Name ADDeploy -Path <path>
```

To install:

```powershell
Install-Module -Name ADDeploy -Scope CurrentUser
```
